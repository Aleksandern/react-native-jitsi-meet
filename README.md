# react-native-jitsi-meet

React native wrapper for Jitsi Meet SDK

## Installation

```sh
npm install react-native-jitsi-meet
```

## Usage

```js
import JitsiMeet from "react-native-jitsi-meet";

// ...

const result = await JitsiMeet.multiply(3, 7);
```
